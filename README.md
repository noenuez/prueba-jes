# JES PRUEBA TECNICA
Clonar repositorio
$ git clone https://github.com/noenuez/prueba-jes.git
# Frontend
para ajustar la direcion de los servicio frontend\src\app\http\index.ts
Construir
## Ir a carpeta
$ cd frontend
## Instalar las dependencia
$ npm install
## Arrancar
levantara en el puerto 3000
$ npm start
# Backend

Construir
## Ir a carpeta
$ cd backend
## Instalar librerias
$ npm install

## Arrancar
$ npm run start
